# My-Statistics-for-Data-Analysis
This repository is a practical collection of statistical methods
used in real-world data analysis.

## Purpose
- Learn statistics through real datasets
- Understand when and why to use each method


## Target Audience
- Data analysts
- Data engineers
- Aspiring data scientists
- Engineers who want to strengthen analytical thinking

## Contents
- Descriptive statistics
- Probability distributions
- Hypothesis testing
- Regression analysis
